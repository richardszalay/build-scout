CCTray Feed

Select this option to add any server that supports the CCTray XML format. This is the most compatible option, but won't provide access to provider-specific functionary (like starting a job) when it becomes available.

Example URL: http://ci.jenkins-ci.org/cc.xml

Hudson/Jenkins

Select this option to add a Hudson or Jenkins server. Include the "jenkins/" or "hudson/" path if your server uses one.

Example URL: http://ci.jenkins-ci.org

TeamCity 6.x

Select this option to add a Team City 6 server.

Warning: Support for TeamCity is experimental and may not yield optimal network performance. This is due to the lack of HTTP compression support and increased HTTP requests required by Team City 6.

Example URL: http://teamcity.codebetter.com
